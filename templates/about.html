About PhishGuard

Understanding our approach to phishing detection

Project Overview

PhishGuard is a final year diploma project designed to help users identify potential phishing websites. Using advanced analysis techniques, our system examines various aspects of a URL to determine if it exhibits characteristics commonly associated with phishing attempts.

This tool is especially relevant in today's digital landscape where phishing attacks are becoming increasingly sophisticated and prevalent. Our goal is to provide a user-friendly interface for quick security assessments of suspicious URLs.

Our Methodology

URL Analysis Features

URL Length

Analyzing URL length as phishing URLs tend to be unusually long

IP Address Usage

Detecting use of IP addresses instead of domain names

Special Characters

Checking for suspicious special characters in URLs

Domain Analysis

Examining domain name characteristics and patterns

Detection Algorithm

Our detection system uses a rule-based approach combined with pattern recognition to identify potential phishing websites. Each URL is analyzed across multiple parameters, with each suspicious characteristic contributing to an overall risk score.

Key Indicators We Check:

  • Presence of IP addresses in URLs
  • Abnormal URL length
  • Use of special characters
  • Multiple subdomains
  • Suspicious redirects
  • SSL/HTTPS usage
